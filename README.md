# 🧪 Quality Portal (SAP UI5)

The **Quality Portal** is a custom SAP UI5 application developed for managing quality inspections, usage decisions, and plant-level reporting. It is deployed on **SAP Business Technology Platform (SAP BTP)** and accessible through the **SAP Launchpad Service**.
